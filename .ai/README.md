# .ai Directory

Назначение: операционный слой для работы ИИ-агента в проекте `gitlab-mcp-server`.

Что здесь хранится:
- `system/` — базовый prompt и правила оркестрации ролей.
- `context/` — бизнес/архитектурный контекст проекта.
- `project/` — карта репозитория и зоны ответственности.
- `standards/` — стиль кода и стандарты тестирования.
- `tasks/` — workflow задач для агента.
- `memory/` — ограничения, решения, допущения, known issues.
- `mcp/` — правила маршрутизации и безопасного использования инструментов.

Связь с текущей документацией:
- Канонические правила проекта остаются в `.ai/system/policy/*` и `AGENTS.md`.
- `.ai/*` задает ежедневный рабочий процесс и структуру исполнения задач.
