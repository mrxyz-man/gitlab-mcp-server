# Handoff Rules

Каждый handoff обязан содержать:
- цель задачи и текущий статус;
- что уже сделано;
- ограничения и риски;
- допущения;
- следующий ожидаемый шаг.

Короткий шаблон:
- From: `{{FROM_ROLE}}`
- To: `{{TO_ROLE}}`
- Task: `{{TASK_ID_OR_TITLE}}`
- Status: `{{STATUS}}`
- Done: `{{DONE_ITEMS}}`
- Risks: `{{RISKS}}`
- Assumptions: `{{ASSUMPTIONS}}`
- Next: `{{NEXT_ACTION}}`
