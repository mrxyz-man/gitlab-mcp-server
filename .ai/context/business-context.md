# Business Context

Проект: MCP-сервер для GitLab (основной клиент: Codex, с возможностью расширения для других ИИ-агентов).

Цели:
- дать агенту безопасный доступ к задачам/label/проектам GitLab;
- обеспечить OAuth-first UX с авто-восстановлением авторизации;
- поддержать multi-instance (gitlab.com + self-hosted GitLab).

Ключевая ценность:
- агент выполняет рабочие сценарии с issue tracker без ручной рутины;
- единый инструмент для локальной разработки и production-использования через MCP.

Нефункциональные ожидания:
- надежность (timeout/retry/ошибки и восстановление);
- безопасность (без утечки секретов);
- предсказуемый DX для конечного пользователя.
