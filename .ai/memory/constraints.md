# Constraints

1. Не хранить секреты/токены в репозитории.
2. MCP tools должны иметь явные контракты входа/выхода.
3. Сетевая интеграция обязана учитывать timeout/retry/error mapping.
4. OAuth flow должен быть безопасен при конкурентных запусках.
5. Решения по внешним библиотекам подтверждать через Context7.
