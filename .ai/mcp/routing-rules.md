# MCP Routing Rules

Порядок по умолчанию:
1. Проверить локальный контекст (`.ai`, `docs`, `src`, `tests`).
2. Использовать filesystem/shell для реализации.
3. Использовать gitlab tools для рабочих операций по задачам.
4. Использовать Context7 только когда локального контекста недостаточно.

Не делать:
- внешние запросы без необходимости;
- разрушительные действия без явного подтверждения задачи.
